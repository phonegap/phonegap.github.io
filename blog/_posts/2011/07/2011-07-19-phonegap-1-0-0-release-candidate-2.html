Hey Everyone,

PhoneGap 1.0.0rc2 has been released today. You can go download it now from http://www.phonegap.com/releasecandidate/.

PhoneGap 1.0.0 is set to officially be released on PhoneGap Day, July 29th, 2011.

Please take a look at our release plan to view a checklist of features that were updated/added. We would greatly appreciate if members of the PhoneGap community can download the release candidate and report any bugs you find on github. Follow the wiki for instructions on how to submit bugs.

Cheers, -Steve Gill
